정규직영입 종료시

[조직소개] 
CS프로덕트실은 구매 이후의 고객 여정과 관련된 개발 영역을 담당하고 있습니다.
우리는 구매 이후의 결제, 배달, 취소/환불 등 우아한형제들 내부에서 관리되는 다양한 도메인의 데이터를 통합하여 고객의 구매 이후의 모든 경험을 책임지고 있습니다.
만약 고객의 경험에 문제가 생기거나 이를 해결해야 할 필요가 생기면 고객의 문제를 즉시 해결할 수 있도록 셀프서비스를 제공하여 고객의 문제를 해결합니다.
또한 셀프서비스를 통해 해결된 고객의 문제에 대하여 고객 만족도 측정하고 이 데이터를 바탕으로 지속적으로 고객 경험 개선을 위한 데이터로 활용합니다.
만약 고객이 스스로 문제를 해결하기 어려운 경우 고객센터의 상담사와 연결할 수 있는 다양한 문의채널(전화, 채팅, 이메일)을 제공하고, 상담사의 업무 효율을 개선하기 위한 시스템도 함께 제공합니다. 

[업무내용]
CS 프로덕트실은 상품, 주문, 결제, 배달, 환불 등 전사의 도메인 데이터를 통합/관리하고 이를 통해 고객/상담사에게 주문 이후 경험을 관리할 수 있는 시스템을 운영합니다.
- 우아한형제들 내부 도메인의 데이터 통합 및 서비스 제공을 위한 CS 게이트웨이 도메인 개발
- 고객에게 다양한 셀프서비스 및 AI 기반의 문의 내역 처리를 위한 고객센터 도메인 개발
- 고객센터의 상담사의 업무 시스템 효율화를 위한 웹 어플리케이션 기반의 고객 상담시스템 개발
- 고객의 전화/채팅 문의 기반의 문의 내역 분석 및 셀프서비스 자동화 서비스 개발 

[지원자격]
- 객체지향 기반의 시스템 아키텍처 설계 및 개발 가능
- Java/Kotlin 및 Spring Framework 개발에 대한 깊은 지식을 바탕으로 Web Application 개발 경험 보유
- 백엔드 및 웹 어플리케이션 관련 개발 5년 이상의 경력
- JPA, Hibernate, 도메인 모델링 등 ORM 사용 경험
- MSA(Micro Service Architecture)의 개념의 이해를 기반으로 개발 및 운영 업무 수행
- Test framework를 활용하여 테스트 코드 작성 경험 보유 

[우대사항]
- 지속적인 코드 품질 개선 및 리팩토링 경험
- 대규 모 분산 시스템에 대한 아키텍처 설계 및 개발 경험
- React, Vue.js 기반의 Frontend 개발 스킬 보유
- 개발팀, 현업 부서와의 적극적인 커뮤니케이션 및 협업 경험
- 애자일 프로세스 기반의 개발 프로세스 경험 보유
- CS(Customer Service) 관련 도메인 지식 및 개발 경험 

[개발환경] 
주요 기술 : Java, Spring Framework(MVC, WebFlux), JPA, Gradle, kafka, Redis, React
저장소 : MySQL, Elasticsearch, Redis
업무 도구 : Git, GitLab, Slack, Jira, Confluence, IntelliJ IDEA Ultimate, Docker, GitHub Copilot
코드 리뷰 / 코드 품질 관리 : GitLab Merge Request, SonarQube
테스트, 빌드, 배포 : JUnit, GitLab CI, Jenkins
모니터링 : Grafana, Prometheus, Loki, OpenTelemetry, Pinpoint
